To use <Abbr>SK</Abbr>AdNetwork, determine the set of networks that have permission to show ads within your app, and check their checkboxes in the Info.plist Generator below.
Then add each network’s `SKAdNetworkIdentifier` list, as shown in concatenated form in the code segment generated by the Info.plist Generator, to your app’s `Info.plist` in your application description <Abbr>XML</Abbr> file inside the `InfoAdditions` tag.

For more information, see [“<Abbr>SK</Abbr>AdNetwork: Configuring the Participating Apps”](https://developer.apple.com/documentation/storekit/skadnetwork/configuring_a_source_app" target="_blank" rel="noopener noreferrer) in Apple’s StoreKit Developer documentation.

If you need a programmatic way to integrate <Abbr>SK</Abbr>AdNetwork identifiers for the AppLovin network (not for any of the other ad networks that AppLovin mediates), for example, in your <Abbr>CI/CD</Abbr>, you can also retrieve that list in the <Abbr>JSON</Abbr> or <Abbr>XML</Abbr> structure that is returned if you request one of the following <Abbr>URL</Abbr>s:

- `https://skadnetwork-ids.applovin.com/v1/skadnetworkids.json` (<Abbr>JSON</Abbr>)
- `https://skadnetwork-ids.applovin.com/v1/skadnetworkids.xml` (<Abbr>XML</Abbr>)

If you also need <Abbr>SK</Abbr>AdNetwork identifiers for other mediated ad networks that you integrated with MAX, check those networks in the Info.plist Generator below.
This will update the `Info.plist` code segment in the Generator by adding each of those networks’ `SKAdNetworkIdentifier` lists.
(If you do not check any networks, only the `SKAdNetworkIdentifier` list from the AppLovin network will appear in the code segment.)

<SKAdNetwork2 />

To leverage the MAX Global <Abbr>SK</Abbr>AdNetwork Report, enable <Abbr>iOS</Abbr> 15 <Abbr>SK</Abbr>AdNetwork reporting by using the app’s `Info.plist`:

- In your app’s `Info.plist`, create a new key named `NSAdvertisingAttributionReportEndpoint` of type `String`.
- Enter the value: `https://postbacks-app.com`.
  
The final result should look something like this:
```xml
⋮
<iPhone>
  <InfoAdditions>
    <![CDATA[
      ⋮
      <key>NSAdvertisingAttributionReportEndpoint</key>
      <string>https://postbacks-app.com</string>
      ⋮
    ]]>
  </InfoAdditions>
</iPhone>
```
<Callout type="tip">
You can check the value of your app’s <Abbr>SK</Abbr>AdNetwork endpoint by using the [Mediation Debugger](../testing-networks/mediation-debugger).
</Callout>

<Callout type="note">
To forward the <Abbr>SK</Abbr>AdNetwork postbacks to one or more third-party endpoints of your choice, please contact your account team.
Note that independent of [this Apple feature](https://developer.apple.com/documentation/bundleresources/information_property_list/nsadvertisingattributionreportendpoint), AppLovin automatically sends a copy of all <Abbr>SK</Abbr>AdNetwork postbacks to the relevant campaign attribution <Glossary>MMP</Glossary>.
</Callout>
