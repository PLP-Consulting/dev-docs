### Banners
```actionscript-3
AppLovinMAX.createBanner(bannerAdUnitId, AdViewPosition.BOTTOM_CENTER);
AppLovinMAX.setBannerPlacement(bannerAdUnitId, «placement»);
```

### MRECs
```actionscript-3
AppLovinMAX.createMRec(mrecAdUnitId, AdViewPosition.CENTERED);
AppLovinMAX.setMRecPlacement(mrecAdUnitId, «placement»);
```

### Interstitials
```actionscript-3
AppLovinMAX.showInterstitial(interstitialAdUnitId, «placement»);
```

### Rewarded Ads
```actionscript-3
AppLovinMAX.showRewardedAd(rewardedAdUnitId, «placement»);
```

<MuteAudio />

```actionscript-3
AppLovinMAX.setMuted(true);  // to mute
AppLovinMAX.setMuted(false); // to unmute
```

<Callout type="warning">
Set the mute state before you load ads.
Some networks (like Google bidding and Google AdMob) return muted or unmuted videos depending on what the mute state is before ad load.
</Callout>

## Enable Verbose Logging
Enable verbose logs with the following code:
```actionscript-3
AppLovinMAX.setVerboseLoggingEnabled(true);
```
