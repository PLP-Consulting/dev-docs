In the AppLovin dashboard you can set defaults for the ad format settings of new (or new and existing) applications:

- Log in to [the AppLovin dashboard](https://dash.applovin.com/).
- Select [<UILabel>AppDiscovery &gt; Monetization &gt; Manage &gt; Applications</UILabel>](https://dash.applovin.com/o/manage).
- Click [<UILabel>Edit Defaults</UILabel>](https://dash.applovin.com/o/manage/defaults) (top right).
- Choose your platform from the <UILabel>Select Platform</UILabel> drop-down list.
- Set your default settings to those you desire.
- Scroll down to the bottom of the page and click <UILabel>âœ…&nbsp;Save Application</UILabel>.
  
If you also want to apply these defaults to your existing apps, check the box at the bottom of the page that is labeled <UILabel>Also apply these settings to all your existing <var>Platform</var> apps</UILabel>.
