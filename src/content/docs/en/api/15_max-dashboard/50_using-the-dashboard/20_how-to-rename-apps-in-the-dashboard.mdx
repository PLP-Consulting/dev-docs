- Log in to [the AppLovin dashboard](https://dash.applovin.com/).
- Select [<UILabel>AppDiscovery &gt; Monetization &gt; Manage &gt; Applications</UILabel>](https://dash.applovin.com/o/manage).
- Click your app. This brings up the <UILabel>Edit Application</UILabel> screen.
- In the <UILabel>Application Name</UILabel> field, type the new name for your app.
- Scroll to bottom of the page and select <UILabel>âœ…&nbsp;Save Application</UILabel>.
