MAX Direct Sold ([<UILabel>MAX &gt; Mediation &gt; Analyze &gt; Direct Sold</UILabel>](https://dash.applovin.com/o/mediation/direct_sold_reporting)) delivers real-time granular data that helps you understand your direct sold campaign performance.
Compare Mode allows you to easily evaluate performance for two different dates.

![Direct Sold Reporting. Select a Saved Report drop-down. Dates. Run Report button. Hide Controls button. Compare Mode toggle. Checkboxes: Day, Hour, Master Publishing Account, Account, Application, Platform, Country (Enter filter value and/or use checkbox to add column), Campaign Order, Campaign, Ad, Ad Format. Table: Platform, Impressions, Clicks, CTR, Estimated Revenue.](/images/direct-sold-reporting-01.png)

After you set up the perfect report you can export it to a `.csv` file or save it for easy access later.

<Callout type="note">Data retention is three months.</Callout>

## Metrics
- <Glossary impressions>Impressions</Glossary>
- Clicks
- <Abbr>CTR</Abbr>
- Estimated Revenue (based on the <Glossary>CPM</Glossary> booked for the campaign)
