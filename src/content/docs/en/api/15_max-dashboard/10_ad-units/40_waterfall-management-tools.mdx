## Ad Unit Manager <Abbr>CSV</Abbr> Tool

Manage and make bulk updates to your waterfall without the need of an <Abbr>API</Abbr> by using the Bulk <Abbr>CSV</Abbr> Ad Unit Manager tool.
Export your current ad units as <Abbr>CSV</Abbr>, make adjustments, and import the file back into MAX.
You can access this feature from the MAX Dashboard by selecting [<UILabel>MAX &gt; Mediation &gt; <Abbr>CSV</Abbr> Tools &gt; Ad Unit Manager</UILabel>](https://dash.applovin.com/o/mediation/ad_unit_manager).
