## <Abbr>iOS</Abbr> 15 Global <Abbr>SK</Abbr>AdNetwork Reporting

Starting with <Abbr>iOS</Abbr> 15, Apple allows developers to send a copy of their <Glossary>SKAdNetwork</Glossary> install <Glossary>postbacks</Glossary> to an endpoint of their choice.
MAX provides a Global <Abbr>SK</Abbr>AdNetwork Report ([<UILabel>MAX &gt; Mediation &gt; Analyze &gt; Global SKA Report</UILabel>](https://dash.applovin.com/o/mediation/global_ska_report)) for developers to access the <Abbr>SK</Abbr>AdNetwork data across all their network partners in one place.
To leverage the MAX Global <Abbr>SK</Abbr>AdNetwork Report, enable <Abbr>iOS</Abbr> 15 <Abbr>SK</Abbr>AdNetwork reporting by using the app’s `Info.plist`:

1. In your app’s `Info.plist`, create a new key named `NSAdvertisingAttributionReportEndpoint` of type `String`.
1. Give that key the value: `https://postbacks-app.com`.
