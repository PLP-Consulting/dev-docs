You can download and integrate the MAX <Glossary>mediation</Glossary> adapters into your app in multiple ways:

Gradle
: AppLovin recommends that you integrate adapters this way.
  AppLovin’s adapters’ <Abbr>POM</Abbr> files automatically integrate the mediated <Glossary>network</Glossary>’s <Abbr>SDK</Abbr> along with its dependencies.
  
Open Source Adapters
: If you require more flexibility in the integration, you can integrate AppLovin’s open source adapters.
  <Glossary>Android</Glossary> <Abbr>SDK</Abbr> v11.0.0 is required in order to compile.
  You are responsible for integrating the mediated <Abbr>SDK</Abbr>s and any dependencies they require.
  You can find these adapters in [the AppLovn-MAX-SDK-Android GitHub repository](https://github.com/AppLovin/AppLovin-MAX-SDK-Android).
  
Manual Integration
: Read [the manual integration instructions](missing-link) if you want to integrate the adapter artifacts, mediated <Abbr>SDK</Abbr>s, and their dependencies manually.
  
<Callout type="warning">
Do not change the adapter’s name or package name.
If you do so, the MAX <Abbr>SDK</Abbr> will not be able to recognize the adapter and ads will not load.
</Callout>
